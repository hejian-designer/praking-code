# UI Source

Recommended editor: Penpot

- Official site: https://penpot.app/
- Why it fits this project:
  - Free and open source
  - Friendly to SVG-based workflows
  - Easier for web UI handoff than locked proprietary formats

Alternative: Lunacy

- Official site: https://icons8.com/lunacy
- Useful if you prefer a desktop app and want simple SVG editing

Files

- `parking-query-mobile-ui.svg`
  - Full mobile page mock for iPhone-size editing
- `parking-query-result-card.svg`
  - Isolated result-card mock for faster iteration

Suggested workflow

1. Open Penpot and create a new file.
2. Import one of the SVG files from this folder.
3. Adjust spacing, size, text, colors, borders, and button proportions.
4. Export the edited result as SVG.
5. Send the SVG back to me.

What to edit first

- Result card height
- Left status badge size
- Right fee box size
- Plate font size
- The spacing between `车主 / 入场时间 / 今日已停`
- Button height, width, and border thickness

If you only want to tweak the card, edit `parking-query-result-card.svg` first.
